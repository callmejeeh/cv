Projet de Codage

Phaser – Création de jeux 2D en HTML5

Objectif

Le projet Phaser est un framework open-source pour créer des jeux 2D interactifs directement dans le navigateur. L’objectif est de combiner programmation web et création de jeux pour produire des expériences ludiques et accessibles.

Technologies utilisées

  • HTML5 : structure et affichage via canvas.
  • CSS3 : styles et animations pour l’interface.
  • JavaScript : logique du jeu et interactions utilisateur.
  • Phaser.js : gestion des scènes, sprites, collisions et entrées.

Fonctionnement

  • Canvas du jeu : espace où tous les éléments sont affichés.
  • Scènes : menus, niveaux, écran de fin de jeu.
  • Sprites et animations : personnages, ennemis, objets animés.
  • Interactions utilisateur : clavier, souris, clics.
  • Mécaniques de jeu : collisions, scores, chronomètres, règles du gameplay.
